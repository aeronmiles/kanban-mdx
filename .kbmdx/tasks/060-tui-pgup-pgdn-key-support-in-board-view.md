Add Page Up and Page Down key support for full-page scrolling in the board view.

## Current state
Go version supports J/K for half-page and ctrl+j/ctrl+k for full-page. Missing standard PgUp/PgDn keys.

## Keys to add
- `PgUp` — Full page scroll up
- `PgDn` — Full page scroll down

kbmdx has these as standard navigation keys in the board view.
